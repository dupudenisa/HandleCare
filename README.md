# HandleCare

Description: 

React JS 

In order for caregivers to have a better sense of what is going on on a daily basis and keep track of every individual more effectively,
as well as sharing  residents progress with their family members at home. 

User Story: 

As a Caregiver: 

I WANT TO track daily health data of a resident 
SO THAT i can have a better sense of what residents need more attention on
SO THAT i can effectively update what I need to in their daily routine. 

AS A family member: 
I WANT TO view daily information on my loved one
SO THAT I can easily track their health and progress
